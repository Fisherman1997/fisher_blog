## 博客主页

##### 安装依赖

```
npm install
//或 yarn install
```

##### 开发环境运行
```
npm run dev
```

##### 生产环境
```
npm run build

```

|   依赖    |          说明          |

|   react   |         基础库         |
|   next    |  项目基于next框架开发  |
